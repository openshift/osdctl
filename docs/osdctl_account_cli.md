## osdctl account cli

Generate temporary AWS CLI credentials on demand

```
osdctl account cli [flags]
```

```

  -h, --help               help for cli
  -o, --output string      Output type
  -p, --profile string     AWS Profile
  -r, --region string      Region
      --verbose            Verbose output
```
